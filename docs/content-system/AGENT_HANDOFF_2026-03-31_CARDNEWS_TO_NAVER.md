## 이번 턴까지 완료된 것

### 1. 블로그 경로

- `Google Blog Only` 워크플로우는 분리 완료 상태다.
- 블로그 메일 발송 경로는 이전에 성공 확인된 상태다.
- 블로그 쪽은 별도 카드뉴스 실패와 분리된 안전 경로로 유지한다.

### 2. 카드뉴스 경로

- 카드뉴스 전용 워크플로우를 별도 분리했다.
- 카드뉴스는 현재 `메일용 시안`까지 성공 확인했다.
- 인스타그램과 페이스북 시안 비율을 분리했다.
  - 인스타그램: `4:5`
  - 페이스북: `1:1`
- 카드 내부의 `HOOK / PROBLEM / DIFFERENCE` 같은 라벨은 최종 시안에 노출하지 않도록 수정했다.

### 3. 쇼룸 의미 고정

- 매우 중요:
  - 여기서 말하는 `쇼룸`은 `파인드가구 홈페이지 안의 온라인 쇼룸`이다.
  - `오프라인 쇼룸 방문`, `내방`, `예약`, `상담 예약`, `직접 와서 보기` 등의 뉘앙스는 금지 방향으로 정리했다.
- 카드뉴스 프롬프트에도 이 의미를 반영했다.

### 4. CTA 방향

- 광고 집행이 아닌 일반 게시물 전제를 반영했다.
- 따라서 `실제 클릭형 CTA 버튼` 전제는 제거했다.
- 카드 마지막 장도 버튼형 문구가 아니라 `온라인 쇼룸에서 더 보기` 계열의 자연스러운 유도 방향으로 조정했다.

## 최종 확인 상태

- n8n 웹훅 호출 성공
- n8n 실행 성공
- Gmail로 카드뉴스 시안 메일 도착 확인
- Gmail 검색 결과에서 최신 시안 메일이 잡히는 것 확인
- 최신 메일 미리보기에서 `쇼룸은 오프라인 전시장이 아니라 ... 홈페이지 ...` 문구 반영 확인

주의:
- Gmail은 같은 제목 스레드로 묶이면서 예전 메일 스니펫이 계속 보일 수 있다.
- 따라서 받은편지함 한 줄 미리보기만 보고 최신 버전 반영 여부를 단정하지 말고, 검색 결과나 최신 시간 기준으로 봐야 한다.

## 현재 카드뉴스 워크플로우 성격

이 워크플로우는 현재 아래 단계까지 성공한 상태다.

1. 입력 payload 수신
2. GPT로 카드뉴스 문구 생성
3. 대표 이미지를 바탕으로 카드형 메일 시안 렌더링
4. Gmail로 시안 발송

아직 하지 않은 것:

- 최종 업로드용 `5장 JPG/PNG 파일` 생성
- 메타 업로드 자동화
- 디자인 완성도 고도화

즉 현재 결과물은 `최종 배포 이미지`가 아니라 `메일로 확인하는 카드뉴스 시안`이다.
